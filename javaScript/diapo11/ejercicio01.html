Une dos o más arrays

var listab = [false, false, false, true, 43]

indexOf()

Busca un elemento en el array y retorna su posición.

join()

Junta los elementos de un array en un String.

lastIndexOf()

Busca la última aparición de un elemento en el array y retorna la posición.

pop()

Quita el último elemento del array y lo retorna.

push()

Añade un nuevo elemento al final del array y retorna su tamaño

reverse()

Invierte el orden de los elementos.

shift()

Quita el primer elemento del array y lo retorna.
